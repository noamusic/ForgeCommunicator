Approval Settings

Require Account Approval

When enabled, new user accounts must be approved by an administrator before they can access workspaces. Users can still sign up, but will see a "pending approval" message until approved.

What happens when enabled:

  • New users can sign up but cannot access workspaces
  • Existing users are automatically considered approved
  • Admins can approve or reject users from the pending approvals page

Require Workspace Creation Permission

When enabled, users need explicit permission to create new workspaces. Users can still join existing workspaces they're invited to.

What happens when enabled:

  • New users cannot create workspaces by default
  • Users can still join workspaces via invite
  • Platform admins can always create workspaces
  • Admins can grant workspace creation permission per user
